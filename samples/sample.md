# Getting Started with docx2md

This is a sample document demonstrating the DOCX to Markdown conversion capabilities.

## Features

The converter supports headings, paragraphs, lists, tables, and images.

## Usage

Simply run: docx2md input.docx output.md

## Conversion Process

The document is parsed into segments, each representing a block-level element.

Each segment is then converted to Markdown using deterministic rules.

## Diagnostics

Any unsupported features are flagged and reported in a diagnostic file.
