# Sample DOCX to Markdown Conversion

This directory contains sample documents for testing the docx2md converter.

## Files

- **sample.docx** - Sample Word document with headings and paragraphs
- **sample.md** - Converted Markdown output
- **sample_diagnostics.md** - Conversion diagnostics report

## Creating Your Own Sample

You can create test documents in Microsoft Word or programmatically using the DocumentFormat.OpenXml library as shown in the test files.
